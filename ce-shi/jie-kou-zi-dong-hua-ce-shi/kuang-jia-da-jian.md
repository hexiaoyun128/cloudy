# 框架搭建

## 工程结构如下

```
├── README.md  //说明文件
├── api_request.py // api请问
├── case
│   ├── test_case.xsd  // 用例约束文件
│   └── xml  //  测试用例
├── config.yml // 配置文件
├── load_case.py  //用例加载
├── login.py // 登录
├── main.py  // 测试入口文件
├── media  // 测试用例使用的文件
├── report  // 报告目录
├── report.py  // 报告生成脚本
├── requirements.txt  //python 依赖
├── test_user.py  // 测试用户
└── utils.py  // 工具函数

```
